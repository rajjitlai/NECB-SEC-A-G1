# NECB-SEC-A-G1 | Team Showcase

A modern, responsive team showcase website built with **Next.js**, **Tailwind CSS (v4)**, and **Framer Motion**. Developed in **2024** at **NIELIT Imphal** under the expert guidance of [Lamzing Technologies](https://www.lamzing.com/).

This project highlights the contributors of the NECB-SEC-A-G1 project with a sleek cyberpunk aesthetic and dynamic GitHub data integration.

## 🚀 Features

- **Modern Dark/Cyberpunk UI**: Glassmorphic effects, neon glow, and grid backgrounds.
- **Dynamic Data**: Fetches contributor profiles (avatars, bios, names) directly from the GitHub API.
- **Fluid Animations**: Smooth entrance, hover, and background animations using Framer Motion.
- **Responsive Design**: Optimized for all screen sizes (Mobile to Ultra-wide).
- **Interactive Cards**: High-performance hover effects and direct links to GitHub profiles.

## 🏁 Getting Started

First, install the dependencies:

```bash
npm install
```

Then, run the development server:

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.
